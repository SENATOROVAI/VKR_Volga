# Система отопления и вентиляции корпуса базы отдыха «Дубрава»

> Выпускная квалификационная работа по направлению **08.03.01 «Строительство»**
> ВлГУ, Муромский институт, кафедра техносферной безопасности · Муром, 2026

Проектирование систем отопления и вентиляции существующего бытового корпуса базы отдыха **«Дубрава»** (п. Мыза, г.о.г. Кулебаки, Нижегородская обл.) при полностью **электрическом** источнике теплоты — газоснабжение на объекте отсутствует.

| | |
|---|---|
| **Объект** | Бытовой корпус 18 × 36 м, кирпичные стены 510 мм, расчётные 1-й и 2-й этажи |
| **Климат** | Расчётный пункт Выкса, температура холодной пятидневки (0,92) **−30 °C** |
| **Состав** | Пояснительная записка (156 с.) + 6 листов графики А1 |
| **Системы** | Водяное отопление, электрокотельная, приточно-вытяжная вентиляция |

---

## Цель

Разработать системы отопления и вентиляции корпуса, обеспечивающие нормируемый микроклимат в санитарно-бытовых помещениях при электрическом источнике теплоты.

**Задачи:**

1. Дать характеристику объекта и выполнить теплотехнический расчёт наружных ограждений.
2. Рассчитать теплопотери здания, разработать систему отопления, выполнить её гидравлический расчёт и подобрать оборудование электрокотельной.
3. Определить нормативный воздухообмен, разработать схему вентиляции, выполнить аэродинамический расчёт и подобрать оборудование.
4. Оценить технико-экономические и энергетические характеристики, обосновать мероприятия по энергосбережению.

---

## Как решали

**1. Характеристика объекта и теплотехника.** Теплотехнический расчёт ограждений выявил существенный дефицит тепловой защиты: фактическое сопротивление теплопередаче кирпичной стены 510 мм составило **0,831** при норме **2,73 м²·°C/Вт** — почти втрое ниже требуемого. Этот результат стал отправной точкой для расчёта нагрузок и для раздела энергосбережения.

**2. Отопление.** Теплопотери рассчитаны через тепловой баланс — потери через ограждения, нагрев инфильтрующегося воздуха и бытовые теплопоступления. Расчётная нагрузка корпуса — **62,4 кВт**. Запроектирована водяная двухтрубная система с насосной циркуляцией и регистрами из гладких стальных труб Ø108×4 (**36 шт.**, теплоотдача 74,4 кВт, запас ~19 %). Гидравлический расчёт выполнен методом удельных потерь давления с увязкой колец.

**3. Источник теплоты.** Подобрана электрокотельная: котёл **ZOTA Solid-X 72** (72 кВт), насос **Grundfos UPS 25-60**, расширительный бак **Reflex N 500**, группа безопасности и узел подпитки с автоматикой.

**4. Вентиляция.** Запроектирована смешанная схема: механический приток **П1 = 3040 м³/ч** (установка Breezart 3700 Lux с электрокалорифером 52,5 кВт), механическая вытяжка **В1 = 1180 м³/ч** (Systemair KT 50-25-4) и естественная вытяжка **ВЕ = 2190 м³/ч**. Воздухообмен определён по нормам, выполнен аэродинамический расчёт; рассмотрен вариант с рекуперацией (КПД ~65 %), снижающий нагрузку на нагрев приточного воздуха с ~49 до ~16 кВт.

**5. Технико-экономическая оценка.** Посчитано годовое электропотребление, эксплуатационные и капитальные затраты, проработаны три мероприятия энергосбережения.

---

## Ключевые результаты

| Показатель | Значение |
|---|---|
| Расчётная нагрузка отопления | **62,4 кВт** |
| Электрокотёл | ZOTA Solid-X 72 (72 кВт) |
| Производительность вентиляции (П1 / В1 / ВЕ) | 3040 / 1180 / 2190 м³/ч |
| Годовое электропотребление | **240,6 тыс. кВт·ч/год** |
| Удельный показатель | 203 кВт·ч/(м²·год) |
| Эксплуатационные затраты (тариф 8,50 руб./кВт·ч) | ~2,04 млн руб./год |
| Потенциал энергосбережения | ~105 тыс. кВт·ч/год (до 44 %) |

**Мероприятия по энергосбережению:** утепление стен (−50,6 тыс. кВт·ч/год), рекуперация тепла вытяжного воздуха (−54,3 тыс. кВт·ч/год, окупаемость 0,7–1,1 года), погодозависимое регулирование (+3–7 %).

---

## Выводы

- Цель достигнута, все задачи доведены до числовых результатов и готового комплекта проектных материалов.
- Разработаны водяная система отопления с электрокотельной и смешанная система вентиляции с подбором оборудования.
- Выявлен и количественно оценён дефицит тепловой защиты стен — главный резерв энергоэффективности объекта.
- Обоснованы три мероприятия энергосбережения с совокупным потенциалом до 44 % базового потребления; рекуперация окупается менее чем за год.
- Электроснабжение исключает местные выбросы загрязняющих веществ.
- Направления развития: наружное утепление до нормы, переход на приточно-вытяжную установку с рекуперацией, уточнение тарифа и числа санитарных приборов на стадии рабочего проектирования.

## Технологический стек

**Нормативная база**
- СП 60.13330.2020 — отопление, вентиляция и кондиционирование
- СП 44.13330.2011 — административные и бытовые здания
- СП 50.13330.2012 — тепловая защита зданий
- СП 131.13330.2020 — строительная климатология
- СП 7.13130.2013 — пожарная безопасность систем ОВК
- ГОСТ 23166-2021, ГОСТ 10704-91, ГОСТ 3262-75; ФЗ № 261 «Об энергосбережении»

**Методики расчёта**
- Тепловой баланс с учётом потерь через ограждения, инфильтрации и бытовых теплопоступлений
- Теплоотдача гладких стальных труб (регистры)
- Гидравлический и аэродинамический расчёт методом удельных потерь давления с увязкой колец
- Источники: Богословский, Сканави (теплоснабжение и вентиляция), Идельчик (справочник по гидравлическим сопротивлениям)

**Подобранное оборудование**
- Котёл ZOTA Solid-X 72 · насос Grundfos UPS 25-60 · бак Reflex N 500
- Приточная установка Breezart 3700 Lux · вытяжной вентилятор Systemair KT 50-25-4

**Графическая часть**
- 6 листов формата А1 в трёх форматах: **DXF** (векторные исходники), **PDF** и **PNG**

> Примечание: конкретное расчётное и чертёжное ПО в пояснительной записке не указано.
